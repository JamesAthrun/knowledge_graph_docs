## 1. 构建方法
### install dependencies
npm install

### serve with hot reload at localhost:8080
npm run dev

### build for production with minification
npm run build

## 2. 页面数据
### 2.1 搜索页
输入关键字得到相关实体搜索列表

### 2.2 实体页
主页面显示出知识图谱，并对于图谱进行可视化编辑
